Key Projects

Hands-on portfolio

Exploration Wells Dashboard – PHE

Dashboard eksplorasi yang menampilkan status teknis ribuan sumur (completed, plugged & abandoned, suspended), distribusi onshore–offshore, serta klasifikasi lingkungan sumur. Data bersumber dari integrasi IHS, CNC, dan sistem internal PHE.

  • Membangun table agregasi di PostgreSQL untuk status well, environment, dan klasifikasi kedalaman.
  • Menggunakan Fusion sebagai UI utama: kartu KPI, grafik batang klasifikasi offshore, pie chart onshore/offshore.
  • Menjaga konsistensi penamaan sumur & contract area agar cocok dengan berbagai sumber data.

Ticketing Data Pipeline & Fraud Analytics – ASDP

Pipeline end-to-end dari data tiket mentah sampai insight operasional: pembersihan data, pemetaan lintasan, RFM, dan fraud detection.

  • Mengolah data tiket dengan Python & SQL, kemudian menyimpannya ke PostgreSQL untuk analitik.
  • Mengembangkan model RFM untuk segmentasi pelanggan dan monitoring penjualan lintasan.
  • Membuat indikator fraud berbasis rule & anomaly patterns dari data transaksi.

Political Social Network Analysis & Campaign Intelligence – Pilpres 2024

Analisis jaringan percakapan publik untuk memetakan klaster dukungan, narasi dominan, influencer, dan pola penyebaran opini pada kampanye Ganjar–Mahfud. Menggunakan Gephi dan text mining untuk membaca tren percakapan serta mengidentifikasi operasi trending dan sentimen terhadap kandidat.

  • Membangun graf interaksi publik (mentions, replies, hashtags) untuk melihat struktur kelompok.
  • Mengidentifikasi klaster narasi: dukungan, kontra-narasi, dan kampanye organik vs terkoordinasi.
  • Memberikan insight strategis berbasis data untuk membaca kekuatan opini publik.

RAG-based Exploration Assistant – PHE

AI · SELF-DEVELOPMENT

Proyek pengembangan diri di bidang AI: prototype LLM assistant berbasis Retrieval-Augmented Generation (RAG) untuk membaca dokumen prognosis & drilling summary, mengekstrak insight geoscience, dan menjawab pertanyaan teknis terkait sejarah pengeboran, target reservoir, dan surface location.

  • Membangun index BM25 untuk pencarian informasi dalam dokumen PDF eksplorasi.
  • Melakukan ekstraksi otomatis (surface location, primary target, rig, reservoir interval, dan metadata well lain) dari teks tidak terstruktur.
  • Menggabungkan retrieval BM25 dengan jawaban naratif berbasis LLM untuk membantu geoscientist memahami dokumen lebih cepat.
